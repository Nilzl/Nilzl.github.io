使用hexo搭建自己的个人博客

发表于 2019-03-09 | 分类于 随笔

偶然间看到使用hexo建立个人博客的方法，觉得挺简单的。正好自己一直以来因为各种原因（好吧就是懒），一直没有去建个人博客。于是乎花了一上午时间，顺带着一起美化了一下window的控制台和PowerShell，总算建立好了自己的这么个小窝。这里就记录一下过程当做第一篇博文吧。

前期准备

要使用hexo，首先需要下载Node.js(下载地址)。
同时需要安装好Git并且在Github上注册自己的账号，最好在创建好私钥和公钥，并且在Github远端配置好公钥。Git相关教程：看这里
准备好这些之后，就可以安装hexo框架了。
